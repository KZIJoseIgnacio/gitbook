# AWS

Amazon Web Services (AWS) es la plataforma de computación en la nube más completa y adoptada del mundo, que ofrece más de 200 servicios integrales mediante centros de datos distribuidos globalmente. Su propósito fundamental es permitir que empresas de cualquier tamaño desde startups hasta agencias gubernamentales puedan obtener potencia de cómputo, almacenamiento de bases de datos y entrega de contenido de forma bajo demanda, eliminando la necesidad de invertir y mantener infraestructura física costosa.

## Servicios de AWS

### Cómputo

* EC2: Crear y ejecutar servidores virtuales en la nube.
* Lambda: Ejecutar código sin administrar servidores (serverless).
* Elastic Beanstalk: Desplegar aplicaciones automáticamente sin gestionar infraestructura.
* Auto Scaling: Ajustar automáticamente la cantidad de servidores según la demanda.
* Lightsail: Crear servidores y hosting simples para proyectos pequeños.

***

### Redes y Entrega de Contenido

* VPC: Crear una red privada virtual dentro de AWS.
* Elastic Load Balancer (ELB): Distribuir tráfico entre varios servidores.
* Route 53: Gestionar dominios y resolver direcciones DNS.
* CloudFront: Entregar contenido rápidamente mediante una red CDN global.
* API Gateway: Crear, publicar y administrar APIs.

***

### Almacenamiento

* S3: Almacenar archivos y datos en la nube.
* EBS: Proveer discos virtuales para instancias EC2.
* EFS: Crear un sistema de archivos compartido entre servidores.
* Glacier: Guardar datos a largo plazo a bajo costo.
* AWS Backup: Automatizar copias de seguridad de recursos.
* DataSync: Transferir grandes volúmenes de datos hacia o desde AWS.

***

### Bases de Datos

* RDS: Administrar bases de datos relacionales en la nube.
* DynamoDB: Ejecutar bases de datos NoSQL rápidas y escalables.
* Aurora: Base de datos optimizada compatible con MySQL y PostgreSQL.
* ElastiCache: Acelerar aplicaciones mediante caché en memoria.

***

### Seguridad, Identidad y Cumplimiento

* IAM: Administrar usuarios, permisos y accesos a AWS.
* Cognito: Gestionar autenticación de usuarios en aplicaciones.
* KMS: Crear y administrar claves de cifrado.
* Shield: Proteger aplicaciones contra ataques DDoS.
* WAF: Proteger aplicaciones web con firewall.
* Inspector: Detectar vulnerabilidades de seguridad en recursos.
* GuardDuty: Detectar amenazas y actividades sospechosas.
* AWS Artifact: Acceder a reportes de cumplimiento legal y normativo.
* AWS System Manager:

***

### Monitoreo y Administración

* CloudWatch: Monitorear métricas, logs y alarmas.
* CloudTrail: Registrar acciones y auditoría de eventos en AWS.
* AWS Config: Evaluar configuraciones y cumplimiento de recursos.
* Trusted Advisor: Recibir recomendaciones para optimizar seguridad y costos.

***

### Infraestructura como Código

* CloudFormation: Crear infraestructura mediante plantillas automatizadas.
* CDK: Definir infraestructura usando código en lenguajes de programación.

***

### Contenedores

* ECS: Ejecutar y administrar contenedores Docker.
* EKS: Ejecutar Kubernetes administrado por AWS.
* Fargate: Ejecutar contenedores sin administrar servidores.

***

### Analítica, Machine Learning e IA

* SageMaker: Crear, entrenar y desplegar modelos de Machine Learning.
* Rekognition: Analizar imágenes y reconocimiento facial.
* Comprehend: Analizar texto y detectar sentimientos.
* Lex: Crear chatbots conversacionales.
* Polly: Convertir texto en voz.
* Translate: Traducir texto automáticamente.

***

### Integración y Mensajería

* SES: Envio de correos simple.
* SNS: Enviar notificaciones por email, SMS o aplicaciones.
* SQS: Crear colas de mensajes para desacoplar sistemas.
* Step Functions: Orquestar flujos de trabajo entre servicios.
* EventBridge: Administrar eventos entre servicios AWS.

***

### Aplicaciones Web y Móviles

* Amplify: Crear aplicaciones web y móviles completas.
* AppSync: Crear APIs GraphQL para aplicaciones.
* Pinpoint: Enviar campañas de marketing y mensajes a usuarios.
* Device Farm: Probar aplicaciones en dispositivos reales.

***

### Migración y Recuperación

* Application Migration Service: Migrar servidores a AWS.
* Application Discovery Service: Analizar infraestructura existente para migrar.
* AWS Migration Evaluator: Estimar costos y beneficios antes de migrar.
* Elastic Disaster Recovery (DRS): Recuperar sistemas ante fallas o desastres.

***

### Testing y Resiliencia

* Fault Injection Simulator (FIS): Simular fallos para probar tolerancia a errores.

***

### Otros Servicios

* Elastic Transcoder: Convertir formatos de video.
* Ground Station: Comunicarse con satélites desde AWS.
* AWS Managed Services: Administrar entornos AWS por AWS.

***

### Arquitectura y Buenas Prácticas

* Well-Architected Tool: Evaluar arquitecturas según mejores prácticas de AWS.
* AWS CAF (Cloud Adoption Framework): Guiar la adopción empresarial de AWS.
