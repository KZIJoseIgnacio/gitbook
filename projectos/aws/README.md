# AWS

Servicios de AWS

1. **EC2 (Elastic Compute Cloud)**: Proporciona capacidad de computación escalable en la nube. Se utiliza para ejecutar aplicaciones y servicios personalizados.
2. **S3 (Simple Storage Service)**: Servicio de almacenamiento de objetos que ofrece durabilidad y escalabilidad para almacenar y recuperar cualquier cantidad de datos.
3. **RDS (Relational Database Service)**: Facilita la configuración, operación y escalado de bases de datos relacionales en la nube.
4. **Lambda**: Ejecuta código de manera automática sin aprovisionar o gestionar servidores. Ideal para aplicaciones sin servidor (serverless).
5. **VPC (Virtual Private Cloud)**: Provee una red virtual aislada donde se pueden desplegar recursos de AWS de forma segura.
6. **IAM (Identity and Access Management)**: Ofrece control de acceso granular a servicios y recursos de AWS mediante la gestión de usuarios y permisos.
7. **CloudFront**: Red de distribución de contenido (CDN) que entrega datos con baja latencia y alta velocidad de transferencia.
8. **DynamoDB**: Base de datos NoSQL totalmente gestionada que proporciona un rendimiento rápido y predecible.
9. **SageMaker**: Permite construir, entrenar e implementar modelos de aprendizaje automático de forma sencilla.
10. **SNS (Simple Notification Service)**: Servicio de mensajería que facilita el envío de notificaciones a múltiples destinatarios.
